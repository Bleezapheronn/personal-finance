# Personal Finance Tracker

## Overview

A cross-platform app to track expenses, manage multiple accounts, sync M-Pesa transactions, and generate useful spending and income reports. Built with a focus on privacy, ease of use, and fast data entry for daily financial habits.[^2][^1]

## Features

- Add expenses and income with details (date, category, account, payment method).[^2]
- Sync transactions from M-Pesa automatically (SMS scraper or API).[^2]
- Assign transactions to categories (Rent, Bills, Investments, etc.), with smart suggestions.[^2]
- View real-time balances for each account and overall.[^2]
- Generate and export reports in Excel or Google Sheets format.[^2]
- Data privacy and security best practices (encryption, offline-first design).[^2]


## Getting Started

1. Clone this repository.
2. Install required dependencies (`npm install` or `yarn install` for JavaScript/React projects).
3. Launch the app (`npm start` or your stack's command).
4. Enter sample data or import from your existing spreadsheet for testing.[^1]
